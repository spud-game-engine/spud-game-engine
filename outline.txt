GAME:
	STAGE... extends [./COLLECTION]:
		SPRITE...:
			SPRITE_INFO:empty
			RENDER_INFO:empty
			RENDER_FRAME=>
				update RENDER_INFO from SPRITE_INFO
				draw
			PHYSICS_INFO:empty
			PHYSICS_FRAME=>
				update PHYSICS_INFO from PHYSICS_INFO
				update SPRITE_INFO from PHYSICS_INFO
		COLLECTION...:
			SPRITE... impliments [../SPRITE]...
			COLLECTION... impliments [../COLLECTION]...
			RENDER_FRAME=>
				call RENDER_FRAME on SPRITE...,COLLECTION...
			PHYSICS_FRAME=>
				call PHYSICS_FRAME on SPRITE...,COLLECTION...
		INPUT...:
			EVENT...=> with INFO
				do something with INFO
			TRIGGER=> with INFO
				call EVENT... with INFO
			PLAY=>
				start listening for INPUT
					on input, call TRIGGER with INFO on INPUT
			PAUSE=>
				stop or ignore input
		PLAY=>
			call PLAY on INPUT...
			call RENDER_FRAME,PHYSICS_FRAME regularally
		PAUSE=>
			call PAUSE on INPUT...
			stop calling RENDER_FRAME,PHYSICS_FRAME regularally
	PLAY=>
		call PLAY on STAGE...
	PAUSE=>
		call PAUSE on STAGE...
